# Laracasts - Command apps with php 

### Comandos

Tornar o arquivo tarefas executável (change mode executable)
```sh 
$ sudo chmod +x ./tarefas
```

Comandos disponíveis

```sh 
$ ./tarefas
```

Opções do comando

```sh 
$ ./tarefas help
```

Executar comando

```sh 
$ ./tarefas [opção]
```

### Banco

 - sqlite
 - create table tarefas(id integer primary key, descricao text not null);
